# 🚀 Apollo GTS - Sistema Full Stack

Este projeto marca o desenvolvimento do sistema **Apollo GTS**, focado na integração real entre interface e lógica de programação.

* **HTML5:** Estrutura e organização visual.
* **JavaScript:** Lógica de processamento e controle do sistema.
* **Git & GitHub:** Controle de versão e publicação profissional.

## ⚙️ O que o código faz
- [x] Cria uma interface de painel de controle.

- [x] Garante a comunicação perfeita entre os arquivos do projeto.
